ChainLit

Chainlit is an open-source Python package that makes it incredibly fast to build and share LLM apps. Integrate the Chainlit API in your existing code to spawn a ChatGPT-like interface in minutes!Think of it like Streamlit, but specifically for assistants/chatbots!
